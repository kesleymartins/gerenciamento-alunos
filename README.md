# gerenciamento-alunos
Projeto desenvolvido como segundo trabalho da matéria Desenvovimento Web 2

##### Configuração
---
```bash
# Setup python venv
python -m venv .venv

# Ativar venv
. .venv/bin/activate

# Instalar dependências
pip install -r requirements.txt

# Realizar migrações
python manage.py migrate

# Abrir Servidor
python manage.py runserver
```
